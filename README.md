# Steam Clone - QAP3 Project

React-based Steam platform clone.

## Tech Stack 
- React 18
- React Router DOM
- Vite
- Context API

## Setup
```bash
cd steem-app
npm install
npm run dev
```

## Git Workflow

- `feature/*` - Work branches
